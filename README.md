# 쇼핑몰 프로젝트

Front-End : ReactJs   
Back-End  : NodeJs(Express)   
   
localhost:3000

## `git clone https://github.com/ShoppingMallProject/ShoppingMall`

저장소 내려받기

## `npm install`

모듈(node_modules) 다운로드

## `npm start`

서비스 실행

## `react-scripts start`

서비스 react 부분만 실행

## `node ./server/server.js`

서비스 nodeJs 부분만 실행
